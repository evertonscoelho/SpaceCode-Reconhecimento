Política de privacidade

Antes de começar a utilizar o Space Code, sugerimos que leia este documento até o fim, onde explicamos como o aplicativo lida com seus dados.

1. Uso da câmera

O Space Code é um aplicativo que utiliza a câmera dos dispositivos para que o usuário possa capturar fotos de peças que são utilizadas no jogo. Estas fotografias são entao enviadas para uma servidor, que não as armazena, mas as processa com o objetivo de identificar as peças presentes na imagem. O algoritmo responsável por esta tarefa é incapaz de identificar qualquer outra coisa na imagem que não sejam as peças do jogo e não armazena informação alguma sobre os resultados obtidos.

Com exceção da câmera, este aplicativo não acessa nenhuma outra funcionalidade do dispositivo e não armazena dados pessoais dos usuários.
